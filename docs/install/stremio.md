# Install Stremio

Stremio is a **media center** for **desktop**, **mobile**, and **web**.

## 1. Create your account

- Open: [Stremio Web](https://web.stremio.com/#/intro)
- Click **Sign Up** and complete the form

## 2. Install the app

- Go to [Stremio Downloads](https://www.stremio.com/downloads) and pick the build for your device (Windows, macOS, Linux, Android, smart TVs, web, and more).
