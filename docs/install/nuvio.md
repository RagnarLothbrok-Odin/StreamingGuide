# Install Nuvio

[Nuvio](https://nuvioapp.space/) is a media player for **Android** and **iOS** that uses the same addons as this guide.

## Android

### 1. Install the app

- **Android TV:** [NuvioTV releases](https://github.com/tapframe/NuvioTV/releases) - download the latest APK and install on your device.
- **Android mobile:** [NuvioMobile releases](https://github.com/tapframe/NuvioMobile/releases) - download the latest APK and install on your device.

### 2. Create your Nuvio account

- Go to [Nuvio sign up](https://nuvioapp.space/account/signup) and follow the account creation steps.

### 3. Prepare your account (addons page)

- Open the [Addons](https://nuvioapp.space/account?tab=addons) tab on Nuvio’s site.
- Remove `Cinemeta` and `OpenSubtitles v3` - this guide provides a metadata and subtitle provider.
- Keep this tab open; you’ll return here to install addons in the final steps.

:::: tip Profiles
Nuvio supports multiple profiles (default: “Profile 1”). You can rename or add more later in settings.
::::

:::: tip Addons
Same addons as the rest of this guide.
::::
